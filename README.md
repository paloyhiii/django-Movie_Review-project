# Django Movie Review Project

A web-based movie review application developed using Django.  
Users can browse movies, view details, and write reviews.

## About This Project

This project was created to practice full-stack web development using Django.  
It covers core concepts such as models, views, templates, URL routing, forms, authentication, and deployment.

---

## Features

- View a list of movies
- View movie details
- Create, update, and delete movies (CRUD)
- Write and delete reviews
- User authentication (Sign up / login / login with github / logout)
- Static files support (CSS, images)
